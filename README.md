# ⚽ 越位 Offside — 給小朋友的越位教學小遊戲

一個用網頁做的互動小遊戲,專門教**六歲左右的小朋友**認識足球的「**越位 (offside)**」規則。
全中文介面 + **注音**標示,圖形和顏色為主、文字很少,看世足轉播時可以陪小朋友一起玩。

> 核心觀念一句話:**傳球的那一刻,你不可以比對方「最後一個守門的人」還靠近球門 —— 不可以偷跑到前面等。**

**線上直接玩**(GitHub Pages):👉 https://hanhanlee.github.io/offside/

## 兩種模式

畫面上方有兩個按鈕可以切換:

### 🎬 看動畫
自動播放幾個情境,讓小朋友先「看懂」越位:

- 🔵 藍色 = 我方　🔴 紅色 = 對方　🟡 黃色 = 守門員 (GK)
- 按 **Pass!**,傳球的瞬間會畫出一條 **offside line**(越位線)。
- 球員在線**後面** → 大大的 **O**(GOAL!,進球歡呼 🎉)
- 偷跑到線**前面** → 大大的 **X**(Offside!,哨聲 📣)
- 按「下一個」看不同情境。

### 🎮 玩遊戲
換小朋友自己操作,分成兩個步驟:

**1️⃣ 傳球(避免越位)**
- 用手指/滑鼠**拖動藍色隊友**。
- 越位線會即時變色:**綠色 = 安全**、**紅色 = 偷跑了**。
- 按 **PASS!**。沒越位才能進入射門;偷跑了會被吹哨、重來。

**2️⃣ 射門(把球踢進去)**
沒越位後,畫面切換成**從球後面看向球門**的視角,換你射門:

- **往你想射的方向拉球** → 箭頭指哪,球就往哪飛(方向 = 你拉的方向)。
- **拉得越長,力道越大**(左邊有力量條)。
- 放開手 = 射門。

射門結果:

| 結果 | 意思 |
|---|---|
| **GOAL!** | 進球!得一顆 ⭐(集滿 5 顆星過關)|
| **SAVED!** | 被守門員撲到了 🧤 |
| **OVER!** | 太大力,飛過橫樑了 |
| **SHORT!** | 太輕,球沒到球門 |
| **POST!** | 打到門柱 🥅 |
| **BLOCK!** | 被擋路的防守球員擋下了 🚧 |

> **障礙物**:如果接球時前方(或左前、右前)有防守球員,他們會出現在射門畫面當**障礙物**擋路 —— 可以**瞄他旁邊的空檔繞過**,或用 **🌈吊高**從他頭頂越過。
>
> **接球位置會影響射門**:在邊路接到球,射門就從那一側出發,角度自然不同;中路接球則正對球門。
>
> **過關**:集滿 5 顆 ⭐ 會有 🏆 **獎盃 + 彩帶撒花**的慶祝動畫!

## 可以調整的選項

**🧤 守門員難度(1 / 2 / 3)**
- **1**:守門員慢慢守,瞄旁邊角落很容易進。
- **2**:守門員快一點、跳得高一點,要瞄好。
- **3**(守護神):又快又會預判,幾乎只能用「吊高球射上角」才進得了。

**🏐 球路(直球 / 普通 / 吊高)**
- ⚡ **直球**:又低又快,守門員來不及反應 → 適合**貼地往角落**轟。
- ⚽ **普通**:中等弧線,最平衡。
- 🌈 **吊高**:高高的彩虹球 → 可以**吊過守門員射上角**(但太大力會飛出去)。

## 小提示(陪玩時可以這樣講)

- 傳球時:「藍色不可以跑到紅線**前面**喔,會偷跑!」
- 射門時:「把球**往你想踢進去的地方拉**,拉長一點比較用力,放手!」
- 遇到守護神:「射**上面的角落**,守門員跳不到那麼高~」
